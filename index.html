Room

  • Create and start room

    First of all, add Branth.start(); in code to start away with purple and blue radial background.

    Next, add const Menu = Branth.Room.add("Menu"); before branth start.

    Explanation: Branth.Room.add(roomName) adds a room with name roomName it will be useful below.

    To start a room, add Branth.Room.start("Menu");

  • Switch between room

    Create multiple room, then just use Branth.Room.start(roomName); to switch between them.

Input

  • Input.Key is a list with size 256

  • Input.Mouse is a list with size 3

  • Input.Touch is a list with size 10

  • Get mouse input

    Branth.Input.Mouse(mouseButtonIndex); -> will return a list of mouse that have boolean properties: .pressed, .released, .hold

    Use Branth.Input.mouseDown(0); to check if left mouse button is pressed (just once when press start, not hold).

    Use Branth.Input.mouseHold(1); to check if middle mouse button is hold.

    Use Branth.Input.mouseUp(2); to check if right mouse button is released.

  • Get keyboard input

    Branth.Input.Key(keyCode);

    Have similar properties to mouse input but using keycode instead of mouse button index.

    Branth.KeyCode stores all the registered keycode that are able to check.

    If Branth.KeyCode doesn't have the code you want, feel free to add it and it will automatically handle.

    Branth.Input.keyDown(Branth.KeyCode.Up); -> Check if arrow key up is pressed (just once when press start, not hold)

    Branth.Input.keyHold(Branth.KeyCode.Shift); -> Check if shift key is hold.

    Branth.Input.keyUp(Branth.KeyCode.Space); -> Check if space key is released.

  • Get touch input (works on touch device only)

    Same, just use touch instead of mouse or key.

    TO note: touch use index like mouse, Branth tracks touch up 10 indexes.

    Even though I never saw once device with 10 touch support.

Sound

  • Play sound

    Play sound by add Branth.Sound.play(name); to code.

    name is like the room name, need to add it first.

    Add sound will be explained in Loader.

  • Loop sound

    Branth.Sound.loop(name);

    To check is sound is playing: Branth.Sound.isPlaying(name);

    To stop sound (not pause, it will reset the playtime to 0): Branth.Sound.stop(name);

  • Set volume

    Branth.Sound.setVolume(name, amount);

    To get volume:

    Branth.Sound.getVolume(name, amount);

OBJ

  • Add and push object

     

    class Chicken extends BranthObject {
	constructor(x, y, size) {
		super(x, y);
		this.size = size;
		this.timer = 0;
		this.eggInterval = 1000;
	}
}

    Use Branth.OBJ.add(name); (OBJ is Branth's object handler)
    to reserve a name for later use to store all following instance.

    Branth.OBJ.add("Chicken");
    ^^^^^^^^^^^^^^^^^^^^^^

    To push an instance to the following name, use:
    Branth.OBJ.push("Chicken", new Chicken(10, 10, 5));

    10, 10, 5 -> are x, y, size, respectively.
    If you look at Chicken class constructor, there is super(x, y); there,
    it's because we extends/implements BranthObject to Chicken.

    BranthObject need x and y. Why do we use branth object?
    because it extends BranthBehaviour where exists start(), update(), and render() method.

    Those 3 methods above are necessary to get OBJ (Branth's object handler) to work.

  • Update and render all object

    To update and render all object is to call all update() and render() function in OBJ lists of instances.

    Add Branth.OBJ.updateAll(); to call all update() for every instance.

    Add Branth.OBJ.renderAll(); to call all render() method for every instance that visible=true.

    visible is a property instantiated as true in BranthObject.
